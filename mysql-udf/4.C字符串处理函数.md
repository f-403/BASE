### 1. memcpy - 内存复制函数
```c
功能
将指定长度的字节从源内存区域复制到目标内存区域。
不关心内存中的内容，可用于复制任何类型的数据（包括字符串、二进制数据等）。
原型

void *memcpy(void *dest, const void *src, size_t n);

dest：目标内存地址（需确保有足够空间）。
src：源内存地址。
n：要复制的字节数。

二进制安全：不会因遇到 \0 而停止复制。
不自动添加字符串终止符：如果复制字符串，需手动添加 \0。

// 复制5个字节（不包含\0）
memcpy(dest, src, 5);
dest[5] = '\0';  // 手动添加终止符
//会把0也复制过去，如果源字符是10位长， 目标是5长度， 第三参数设置为5，它最多复制5个，但不会加\0， 
// 如果源字符是3， 目标长度是5, 设置长度是5， 它会把源字符3的结尾\0也会复制过去。
// 最好以第三个参数为下标添加\0字符
```

### 2. snprintf - 安全的格式化输出函数
```c
功能
将格式化的数据写入字符串缓冲区，并确保不会溢出。
自动添加字符串终止符（如果空间足够）。
原型
int snprintf(char *str, size_t size, const char *format, ...);
参数
str：目标字符串缓冲区。
size：缓冲区的最大容量（包括终止符 \0）。
format：格式化字符串，后跟可变参数。
返回值
成功：返回原本应写入的字符串长度（不包含终止符）。
如果返回值 ≥ size，说明输出被截断。
特点
安全防溢出：最多写入 size-1 个字符，自动添加 \0。
支持格式化输出：如 %d、%s 等占位符。

#include <stdio.h>

int main() {
    char dest[10];
    int len;

    // 尝试写入超长字符串
    len = snprintf(dest, sizeof(dest), "Hello, World!");

    printf("写入的字符串: %s\n", dest);  // 输出: Hello, W
    printf("应写入的长度: %d\n", len);  // 输出: 13
    printf("实际写入的长度: %zu\n", strlen(dest));  // 输出: 9

    return 0;
}
```

### strncpy_s
```c
int main() {
    char src[] = "Hello";
    char dest[3];

    strncpy_s(dest, src, 2);//最大不能超过目标数组-1,
    //它要在后面写\0, 如果不留一个字符给它写0会程序终止
    printf("dest: %s\n", dest);
    return 0;
}
```
